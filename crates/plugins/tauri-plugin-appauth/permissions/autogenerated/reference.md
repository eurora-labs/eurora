## Default Permission

Run OAuth 2.0 / OIDC authorization on mobile via AppAuth.

#### This default permission set includes the following:

- `allow-discover`
- `allow-authorize`
- `allow-authorize-browser-only`
- `allow-refresh`
- `allow-end-session`
- `allow-subscribe-events`
