## Default Permission

Drive the native Sign in with Apple sheet (ASAuthorizationController) on iOS.

#### This default permission set includes the following:

- `allow-sign-in-with-apple`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

</td>

</td>
</tr>

<tr>
