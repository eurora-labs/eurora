# Eurora

Welcome to Eurora, a context aware, cross-platform, fully private and open source AI platform.

The primary goal of Eurora is to provide seamless OS integration for AI assistants and ensure full privacy for users all around the world by applying the highest standards set by the European Union. We have first-party support for local deployment and will extensive documentation for running this on your own home server.

We're a community driven project, and we welcome contributions from anyone who is interested in helping us build a better future for AI.

## Installation
We support Windows, macOS, and Linux (x11). The installation will be available soon.
